El método estadístico


¿Qué pasos se deben seguir para realizar un estudio estadístico? Una vez definida la variable a estudiar, lo que será el eje central que articulará todo lo demás, y que responde a las preguntas: ¿Qué necesito estudiar? ¿por qué? y ¿para qué? Diseñaremos el experimento...


Experimento

Esta fase responde a la siguiente pregunta:¿Cómo lo voy a hacer? Aquí lo importante es definir la metodología, ya que existen diferentes modos y consideraciones. Así pues, debemos establecer el tipo de muestreo, el tamaño de la muestra, el modo de recolección de datos (por ejemplo, a través de bases de datos o de encuestas personalizadas, en persona, por internet o por teléfono, etc.)

1. Recopilación de datos

Una vez que hemos planteado el problema debemos recoger los datos, aplicando el plan de acción definido en el experimento.

2. Organización de los datos

Una vez que tenemos todos los datos recogidos, hay que unificarlos y organizarlos. Para esto, existen diversos programas o plataformas que permiten la tabulación y luego nos permite calcular determinadas medidas para analizar, como lo es Excel, GeoGebra, etc. La intención de este paso es la de sintetizar la información y exponerla de una forma más ordenada para facilitar su análisis.

3. Presentación de datos

La presentación de datos debe hacerse de una forma más amigable para los destinatarios y para eso, la mejor manera es hacerlo a través de gráficos. Existen diferentes tipos de gráficos estadísticos, la elección del mismo será criterio del estadista, seleccionando el más adecuado para visibilizar los resultados de la mejor manera posible. Para mas información, click en las imagenes

4. Análisis e interpretación de datos

Por último, y lo más importante, tenemos el análisis e interpretación de los datos. El fin de la estadística es poder predecir y tomar buenas decisiones, y esto será posible con un buen análisis, de nada sirve realizar todas las fases del proceso estadístico correctamente si al final la interpretación es errónea. Si esto sucediera, entonces las decisiones no serán buenas y el estudio tendrá un efecto no deseado.
